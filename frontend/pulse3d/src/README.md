## Getting Started

To run the development server:

```bash
npm i
npm run dev
```

Node version used:
Node==17.6.0

E2E testing: (two terminals)

```bash
npm run dev
```

The following command opens intractive testing window

```bash
npm run cy:open-only
```

Unit testing:

```bash
npm run test:unit
```

Generate static files: (/out)

```bash
npm run export:prod
npm run export:test
```
